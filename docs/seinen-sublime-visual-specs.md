# SECTION 01: CORE VISUAL IDENTITY

## 1.1 Style Definition

**Seinen Sublime Atmosférico** is a hybrid visual style fusing:

| Element | Source | Visual Expression |
|---------|--------|-------------------|
| **The Sublime** | Romantic painting tradition | Vast scale, atmospheric dissolution, cosmic terror |
| **Seinen Aesthetic** | Japanese animation/manga | Precise lineart, detailed decay, cinematic composition |
| **Dark Fantasy** | Gothic/Post-apocalyptic | Ruined architecture, organic corruption, melancholic beauty |

**Core Principle:** *Describe the infinite with exact strokes* — cosmic horror rendered with surgical precision.

## 1.2 Visual Pillars

```
┌─────────────────────────────────────────────────────────────────┐
│                    FOUR VISUAL PILLARS                          │
├─────────────────────────────────────────────────────────────────┤
│  1. DEPTH LAYERING        │  2. PRECISE DECAY                   │
```

# SECTION 03: DEPTH & ATMOSPHERE SYSTEM

## 3.1 Three-Layer Depth Model

```
VIEWER
   │
   ▼
┌──────────────────────────────────────────────────────────────┐
│  LAYER 1: FOREGROUND — Sharp Focus                           │
│  ─────────────────────────────────────────────────────────── │
│  • Distance: 0-3 meters                                      │
│  • Focus: 100% sharp                                         │
│  • Detail: Maximum (textures, materials, small objects)      │
│  • Color: Full saturation, highest contrast                  │
│  • Lineart: Crisp, 1-2px weight, full opacity               │
├──────────────────────────────────────────────────────────────┤
│  LAYER 2: MIDGROUND — Soft Transition                        │
│  ─────────────────────────────────────────────────────────── │
│  • Distance: 3-15 meters                                     │
│  • Focus: 60-80% (subtle blur)                              │
│  • Detail: Reduced (shapes recognizable, textures fade)      │
│  • Color: Desaturated 20-30%, lower contrast                │
│  • Lineart: Softer, 0.5-1px, 70% opacity                    │
│  • Fog: 15-30% opacity overlay                              │
├──────────────────────────────────────────────────────────────┤
│  LAYER 3: BACKGROUND — Atmospheric Blur                      │
│  ─────────────────────────────────────────────────────────── │
│  • Distance: 15+ meters                                      │
│  • Focus: 20-40% (significant blur)                         │
│  • Detail: Minimal (silhouettes, masses, suggestions)        │
│  • Color: Highly desaturated, near-monochrome               │
│  • Lineart: None or barely visible, 30% opacity             │
│  • Fog: 40-70% opacity overlay                              │
└──────────────────────────────────────────────────────────────┘
   │
   ▼
INFINITE VOID (implied, never shown clearly)
```

## 3.2 Volumetric Fog Specifications

### Fog Density Zones

| Zone | Distance | Opacity | Blur | Color |
|------|----------|---------|------|-------|
| Clear | 0-2m | 0-5% | None | N/A |
| Light Mist | 2-5m | 5-15% | 2px | `#8a9199` |
| Medium Fog | 5-15m | 15-35% | 5-10px | `#8a9199` → `#b8bcc2` |
| Dense Fog | 15-30m | 35-60% | 15-25px | `#b8bcc2` |
| Obscuring | 30m+ | 60-90% | 30px+ | `#b8bcc2` → `#8a9199` |

### Fog Behavior
- **Movement:** Slow, drifting horizontally (0.5-2% viewport per second)
- **Layers:** Minimum 3 fog layers at different depths/speeds
- **Edge quality:** Soft, feathered (never hard edges)
- **Vertical gradient:** Denser at bottom, lighter at top

### Fog Rendering Technique
```
Layer 1 (Far):    Gaussian blur 20px, opacity 30%, speed 0.5%
Layer 2 (Mid):    Gaussian blur 10px, opacity 40%, speed 1%
Layer 3 (Near):   Gaussian blur 5px,  opacity 20%, speed 1.5%
```

## 3.3 Atmospheric Perspective

### Value Shift Per Layer
```
Foreground:  Black point: #050608  White point: #e8e4dc
Midground:   Black point: #1a2633  White point: #b8bcc2
Background:  Black point: #2d3d4a  White point: #8a9199
Deep BG:     Black point: #2d3d4a  White point: #5a6570
```

### Saturation Decay
- Foreground: 100% base saturation
- Per 10 meters: -15% saturation
- Background: Near monochromatic (10-20% saturation)

# SECTION 04: LIGHTING SYSTEM

## 4.1 Primary Light Sources

### Volumetric God Rays
```
┌─────────────────────────────────────────┐
│             LIGHT SOURCE                │
│                  │                      │
│          ┌──────┴──────┐                │
│         ╱               ╲               │
│        ╱                 ╲              │
│       ╱    FOG MEDIUM     ╲             │
│      ╱                     ╲            │
│     ╱   VISIBLE RAYS        ╲           │
│    ╱   (scattered light)     ╲          │
│   ╱                           ╲         │
│  ▼                             ▼        │
│ GROUND (partial illumination)           │
└─────────────────────────────────────────┘
```

**Specifications:**
- Cone angle: 15-45°
- Edge softness: High (feathered 30-50%)
- Internal variation: Streaked (dust particles)
- Color: `#b8bcc2` to `#e8e4dc` (never pure white)
- Opacity: 3-8% (subtle, not overpowering)

### Ambient Occlusion
- **Intensity:** High (dark corners are DARK)
- **Radius:** Medium-large (15-30px equivalent)
- **Color:** `#050608` (pure void black)
- **Application:** All recesses, under objects, corners

### Rim/Back Lighting
- **Purpose:** Separate figures from background
- **Intensity:** Subtle (10-20% brightness boost)
- **Width:** 1-3px edge highlight
- **Color:** Cool (`#8a9199`) or match fog color

## 4.2 Light Types & Emotional Function

| Light Type | Technical Spec | Emotional Function |
|------------|----------------|-------------------|
| **Volumetric rays** | Cone, 5-8% opacity, visible in fog | Partial revelation, hope corrupted |
| **Backlight** | Silhouette-creating, high contrast | Hidden threat, unknowable |
| **Grazing/Raking** | Low angle, exaggerates texture | Time's damage, decay visible |
| **Bioluminescence** | Localized glow, unnatural colors | Wrong life, corruption |
| **Absence** | Pure black, no reflection | Active void, presence in nothing |

## 4.3 Shadow Specifications

### Shadow Types
```
CAST SHADOWS
├── Softness: Medium-soft (10-20px feather)
├── Color: #0a0c0f (not pure black)
├── Opacity: 70-90%
└── Never pure black unless void

FORM SHADOWS
├── Transition: Gradual (5-10% value steps)
├── Core shadow: Slightly cooler than base
└── Reflected light: Minimal (5-10% bounce)

AMBIENT OCCLUSION
├── Color: #050608 (void black)
├── Intensity: High
└── Radius: Large
```

# SECTION 05: LINEART & RENDERING

## 5.1 Line Weight System

### By Distance
| Layer | Line Weight | Opacity | Style |
|-------|-------------|---------|-------|
| Foreground | 1.5-2.5px | 100% | Crisp, confident |
| Midground | 0.75-1.5px | 60-80% | Slightly soft |
| Background | 0-0.75px | 20-40% | Fading, broken |

### By Importance
| Element | Line Weight | Notes |
|---------|-------------|-------|
| Character outline | 2-3px | Heaviest weight |
| Character detail | 1-1.5px | Internal lines |
| Architecture major | 1.5-2px | Structural edges |
| Architecture detail | 0.5-1px | Surface detail |
| Organic/fog | 0-0.5px | Minimal or none |

### Line Quality
- **Foreground:** Clean, confident, varied pressure
- **Decay elements:** Broken, scratchy, interrupted
- **Organic:** Soft, tapered, flowing
- **Fog/atmosphere:** No lineart (shape only)

## 5.2 Rendering Style: Hybrid Cel-Shading

### The Hybrid Approach
```
FOREGROUND ELEMENTS          BACKGROUND/ATMOSPHERE
─────────────────────────    ─────────────────────────
Hard cel-shading             Soft digital painting
├── 2-3 value steps          ├── Smooth gradients
├── Clear shadow edges       ├── No distinct edges
├── Visible lineart          ├── No lineart
└── High detail              └── Impressionistic
```

### Value Steps (Cel-Shading)
```
LIGHT           ████████████████████  #e8e4dc (Bone)
                         │
HALFTONE        ████████████████████  #b8bcc2 (Fog Light)
                         │
SHADOW          ████████████████████  #5a6570 (Mid Gray)
                         │
CORE SHADOW     ████████████████████  #2d3d4a (Metal Blue)
                         │
OCCLUSION       ████████████████████  #0a0c0f (Deep Black)
```

## 5.3 Material Rendering

### Metal (Rusted)
```
Base:           #2d3d4a (Metal Blue)
Rust patches:   #a65d3f → #6b3d2a (varied)
Highlights:     #8a9199 (desaturated)
Deep corrosion: #050608 (void)
Texture:        Pitted, scaled, flaking
Pattern:        Follows water flow, gravity
```

### Concrete (Decayed)
```
Base:           #5a6570 (neutral gray)
Cracks:         #0a0c0f (dark voids)
Staining:       #6b3d2a (rust runoff)
Exposed rebar:  #a65d3f (rust)
Moss/growth:    #2d3d4a (dark, unhealthy)
Texture:        Rough, porous, crumbling
```

### Organic (Corrupted)
```
Skin/flesh:     #8a9199 → #b8bcc2 (pale, bloodless)
Wounds:         #6b3d2a → #a65d3f (oxidized blood)
Bone:           #e8e4dc (stark white)
Growth/fungus:  #1a2633 (dark, spreading)
Texture:        Wet, membranous, wrong
```

### Fabric (Weathered)
```
Base:           Desaturated version of original color
Wear patterns:  Lighter at edges, folds
Tears:          Dark interior, frayed edges
Wet areas:      Darker, higher saturation
Movement:       Follows wind/gravity believably
```

## 6.2 Camera Angles

### Dutch Angle (Tilted Horizon)
- **Tilt:** 5-15° from horizontal
- **Use:** Reality distortion, psychological unease
- **When:** Perception-warping moments, madness

### Extreme Low Angle
- **Camera:** Ground level or below
- **Use:** Make structures/entities towering
- **When:** Encountering the vast, divine presence

### Extreme High Angle
- **Camera:** Far above subject
- **Use:** Diminish figure, show insignificance
- **When:** Cosmic perspective, isolation

### Dutch + Low Combination
- **Effect:** Maximum disorientation
- **Use sparingly:** Only at peak psychological moments

## 6.3 Aspect Ratios

| Ratio | Use | Emotional Effect |
|-------|-----|------------------|
| **2.39:1** (Cinematic) | Horizontal vastness | Oppressive width, lonely |
| **16:9** (Standard) | General use | Balanced |
| **4:3** (Classic) | Claustrophobic interiors | Boxed in, trapped |
| **9:16** (Vertical) | Towering heights | Overwhelming scale |
| **1:1** (Square) | Focused portraits | Intimate, confrontational |

# SECTION 07: CHARACTER DESIGN

## 7.1 Figure Rendering

### Proportions
- Base: Realistic with subtle stylization
- Limbs: Slightly elongated (105-110%)
- Posture: Often hunched, defensive, weary
- Silhouette: Must read clearly against any background

### Silhouette Priority
```
CLEAR SILHOUETTE CHECKLIST:
□ Figure readable as black shape on white
□ Key gesture visible in outline
□ No tangent lines with background
□ Negative space defines form
□ Distinct from environment shapes
```

### Expression
- **Eyes:** Often hidden by shadow, hair, or hood
- **When visible:** Exhausted, haunted, hollow
- **Glowing eyes:** Reserve for possessed/corrupted/divine

### Clothing
- Layers: Multiple, worn, practical
- Material: Heavy fabrics, leather, metal accents
- Condition: Weathered, repaired, stained
- Movement: Responds to environment (wind, moisture)

## 7.2 The Corrupted/Possessed

### Visual Indicators of Perception Distortion

**What the viewer sees (corrupted perception):**
```
┌─────────────────────────────────────┐
│                                     │
│    MONSTROUS APPEARANCE             │
│    ├── Glowing eyes (unnatural)     │
│    ├── Distorted proportions        │
│    ├── Visible decay/corruption     │
│    ├── Threatening posture          │
│    └── Dark aura/particles          │
│                                     │
└─────────────────────────────────────┘
```

**What is real (true form):**
```
┌─────────────────────────────────────┐
│                                     │
│    ORDINARY HUMAN                   │
│    ├── Normal eyes (fearful)        │
│    ├── Standard proportions         │
│    ├── Dirt, wounds (mundane)       │
│    ├── Defensive posture            │
│    └── No supernatural elements     │
│                                     │
└─────────────────────────────────────┘
```

### Dual-State Design
- Design characters with BOTH versions
- Corrupted version: Exaggerated features, wrong colors
- True version: Pathetic, human, sympathetic
- Transition: Glitch effects, double exposure, partial reveal

## 7.3 Divine/Cosmic Entities

### The Rule of Never-Fully-Shown
```
ALLOWED:
├── Partial silhouettes
├── Reflected images (distorted)
├── Effects of presence (environment warping)
├── Peripheral glimpses
└── Symbolic representations

FORBIDDEN:
├── Full clear view
├── Defined anatomy
├── Understandable form
└── Eye contact with viewer
```

### Visual Indicators of Presence
- **Environmental:** Temperature shift (frost/heat), light bending
- **Physical:** Nosebleeds, trembling, shadow distortion
- **Spatial:** Perspective breaks, scale inconsistency
- **Auditory (implied):** Characters covering ears, birds fleeing

# SECTION 08: ENVIRONMENT DESIGN

## 8.1 Architectural Language

### Ruined Industrial
```
ELEMENTS:
├── Massive scale (dwarfs humans)
├── Repeating structural elements
├── Broken symmetry
├── Exposed internals (pipes, wires, rebar)
├── Rust + concrete dominant materials
└── Purpose unclear or forgotten

REFERENCE: Blame!, NieR:Automata, Stalker
```

### Organic Corruption
```
ELEMENTS:
├── Biological growth on inorganic surfaces
├── Membrane-like structures
├── Pulsing/breathing suggestion
├── Wrong colors (dark where light should be)
├── Spreading patterns
└── Neither fully alive nor fully dead

REFERENCE: Annihilation, Berserk Eclipse, Bloodborne
```

### Liminal Spaces
```
ELEMENTS:
├── Threshold architecture (doors, bridges, gates)
├── Transitional spaces (hallways, stairs, platforms)
├── Neither inside nor outside
├── Unclear destination
├── Repetitive but slightly wrong
└── Empty of expected inhabitants

REFERENCE: Liminal space photography, Backrooms
```

## 8.2 Environmental Storytelling

### Decay Patterns Tell History
```
RUST PATTERNS:
└── Follow gravity (drip lines)
└── Follow water flow
└── Concentrate at joints/seams
└── Spread from damage points

CRACK PATTERNS:
└── Radial = impact from point
└── Parallel = structural stress
└── Web = age/settling
└── Directional = force applied

GROWTH PATTERNS:
└── Follow moisture
└── Avoid certain areas (why?)
└── Concentrate at death
└── Spread from origin point
```

### Signs of Former Habitation
- Personal objects (weathered, scattered)
- Attempted repairs (failed)
- Defensive modifications (breached)
- Messages (faded, cryptic, desperate)

### Dust/Ash
```
Count:      100-500 particles
Size:       0.5-2px
Color:      #8a9199 (fog gray)
Opacity:    20-40%
Movement:   Drift with wind direction
Speed:      1-3 px/frame
Lifetime:   Medium
```

### Fog Wisps
```
Count:      10-30 wisps
Size:       Large, amorphous
Color:      #8a9199 @ 10-30% opacity
Movement:   Slow horizontal drift
Speed:      0.2-1% viewport/second
Blur:       High (20-40px)
```

## 9.2 Post-Processing Stack

### Recommended Order
```
1. BASE RENDER
      ↓
2. DEPTH OF FIELD
   └── Focus: Foreground subject
   └── Blur: Gradual increase with distance
      ↓
3. VOLUMETRIC FOG LAYERS
   └── 3 layers minimum
   └── Different depths/speeds
      ↓
4. COLOR GRADING
   └── Desaturate: -20-30%
   └── Cool shift: +10-15%
   └── Contrast: Slight increase
      ↓
5. VIGNETTE
   └── Opacity: 30-50%
   └── Size: 70-80% of frame clear
   └── Color: #050608 (void black)
      ↓
6. FILM GRAIN
   └── Intensity: 3-8%
   └── Size: Fine (1-2px)
   └── Animated: Yes (subtle)
      ↓
7. CHROMATIC ABERRATION (optional)
   └── Intensity: Very subtle (1-2px)
   └── Location: Edges only
      ↓
8. FINAL OUTPUT
```

# SECTION 10: ANIMATION GUIDELINES

## 10.1 Atmospheric Animation

### Fog Movement
```
Layer 1 (Far):
├── Direction: Left to right
├── Speed: 0.3% viewport/second
├── Vertical drift: ±2% oscillation over 30s
└── Opacity pulse: ±5% over 20s

Layer 2 (Mid):
├── Direction: Right to left (counter)
├── Speed: 0.6% viewport/second
├── Vertical drift: ±3% oscillation over 25s
└── Opacity pulse: ±8% over 15s

Layer 3 (Near):
├── Direction: Left to right
├── Speed: 1% viewport/second
├── Vertical drift: ±1% oscillation over 20s
└── Opacity pulse: ±3% over 25s
```

### Light Ray Pulse
```
Intensity oscillation: ±30% from base
Cycle duration: 6-10 seconds
Curve: Smooth sine wave
Never fully off, never harsh
```

### Particle Float
```
Primary direction: Upward
Secondary: Horizontal drift (wind)
Speed variation: ±50% between particles
Rotation: Slow tumble (optional)
Spawn: Continuous, random distribution
```

## 10.2 Character Animation Principles

### Movement Quality
- **Weight:** Heavy, deliberate, tired
- **Speed:** Slower than typical anime (70-80% speed)
- **Pauses:** Frequent micro-pauses in movement
- **Breathing:** Visible, irregular, labored

### Environmental Response
- Hair/clothing respond to wind/atmosphere
- Footsteps disturb dust/water
- Breath visible in cold
- Body language reflects unseen threat awareness

## 10.3 Revelation Moments

### The Glimpse
```
FRAMES 1-10:   Fog moves normally
FRAMES 11-13:  Fog thins slightly (5-10%)
FRAMES 14-16:  SHAPE VISIBLE (silhouette only)
FRAMES 17-20:  Fog returns
FRAMES 21+:    Normal, but viewer knows
```

### The Approach
```
Background entity:
├── Frame 1: Not visible
├── Frame 100: 1% opacity, far blur
├── Frame 200: 3% opacity, still blurred
├── Frame 300: 5% opacity, shape discernible
├── Frame 400+: No clearer (never resolves)
└── Tension from non-resolution
```

# SECTION 13: QUALITY CHECKLIST

## 13.1 Pre-Production Checklist

```
□ Three-layer depth plan defined
□ Primary focal point identified
□ Color palette limited to specification
□ Light source(s) determined
□ Fog density zones mapped
□ Scale relationships established
□ Emotional beat identified
```

## 13.2 Production Checklist

```
COMPOSITION
□ Horizon obscured
□ Figure silhouette reads clearly
□ Foreground anchor present
□ Negative space purposeful
□ Scale communicates intended emotion

ATMOSPHERE
□ Minimum 3 fog layers
□ Fog density increases with distance
□ Particles present (rust/dust)
□ Light interacts with atmosphere
□ Vignette applied

COLOR
□ Palette matches specification
□ Rust accent used as focal point
□ Saturation decreases with distance
□ No pure black (except void)
□ No pure white (except bone details)

RENDERING
□ Foreground lineart crisp
□ Background lineart faded/absent
□ Cel-shading on figures
□ Soft painting on atmosphere
□ Material textures appropriate to decay

DETAIL
□ Decay patterns follow logic (gravity, water, stress)
□ Materials specifically named (not generic "metal")
□ Evidence of history/story
□ No detail in deep background
□ Maximum detail in focal area
```

## 13.3 Final Review Questions

```
1. Can you identify three distinct depth layers?
2. Is the focal point immediately clear?
3. Does the atmosphere feel thick and present?
4. Is the scale relationship emotionally effective?
5. Do decay patterns tell a story?
6. Is anything too clearly defined that should be mysterious?
7. Is anything too vague that should anchor the viewer?
8. Does the image create the intended emotional response?
```
